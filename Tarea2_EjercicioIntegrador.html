TAREA 2 - JavaScript

Ejercicio Integrador

A continuacion podemos encontrar el código de un supermercado que vende productos.

  • una clase Producto que representa un producto que vende el super
  • una clase Carrito que representa el carrito de compras de un cliente
  • una clase ProductoEnCarrito que representa un producto que se agrego al carrito
  • una función findProductBySku que simula una base de datos y busca un producto por su sku

El código tiene errores y varias cosas para mejorar / agregar

Ejercicios

1) Arreglar errores existentes en el código

  1. Al ejecutar agregarProducto 2 veces con los mismos valores debería agregar 1 solo producto con la suma de las cantidades.
  2. Al ejecutar agregarProducto debería actualizar la lista de categorías solamente si la categoría no estaba en la lista.
  3. Si intento agregar un producto que no existe debería mostrar un mensaje de error.

2) Agregar la función eliminarProducto a la clase Carrito

  1. La función eliminarProducto recibe un sku y una cantidad (debe devolver una promesa)
  2. Si la cantidad es menor a la cantidad de ese producto en el carrito, se debe restar esa cantidad al producto
  3. Si la cantidad es mayor o igual a la cantidad de ese producto en el carrito, se debe eliminar el producto del carrito
  4. Si el producto no existe en el carrito, se debe mostrar un mensaje de error
  5. La función debe retornar una promesa

3) Utilizar la función eliminarProducto utilizando .then() y .catch()

Los resultados del ejercicio pueden verse en la Consola
